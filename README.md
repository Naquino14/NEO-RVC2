# *N*ew - *R*emote *V*ehicle *C*ommand and *C*ontrol

Command and monitor a vehicle from a handheld LoRa node (FOB). Fob Hardware: [Heltec Automation WiFi LoRa 32(V3)](https://heltec.org/project/wifi-lora-32-v3/)

The tracker and controller (TRC) sits in the vehicle and interfaces with the car and its peripherals. 

This is a work in progress project. 

Planned features:
* Gnss tracking
* 1-2 layer encryption (AES outside & possibly ADIS inside)
* OBDII diagnostics
* Remote start/stop
* FUOTA
* Warning speaker and a master caution/mute button panel

Feature backburner:
* K-Ka monitor attachment
* BLE Android app to inteface with the FOB
* Shift light controller
